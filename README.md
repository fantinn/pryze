# PryzeGear

Site institucional da **PryzeGear**, desenvolvido para apresentar a marca, seus produtos e identidade visual de forma moderna, rápida e responsiva.

## 🌐 Acesse
https://pryzegear.com.br

## 📌 Sobre o projeto
Landing page focada em:
- apresentação da marca  
- exibição de produtos  
- design moderno e minimalista  
- navegação simples e direta  

## ⚙️ Tecnologias
- HTML5  
- CSS3  
- JavaScript  
- GitHub Pages (deploy)

## 📱 Responsividade
O site foi desenvolvido para funcionar bem em:
- Desktop  
- Tablet  
- Mobile  

## 🚀 Objetivo
Criar uma presença digital profissional para a marca **PryzeGear**, destacando produtos e identidade visual com performance e boa experiência do usuário.

## 📄 Licença
Projeto desenvolvido para fins de apresentação e portfólio.
